Platform

Built for the moment when annotation becomes infrastructure.

The product starts with a request, turns it into an auditable annotation project, routes work across AI and human reviewers, then exports clean rows with labels, rationales, confidence, adjudication, and source evidence.

01

Agent-planned projects

Natural-language intake becomes a schema, rubric, examples, QA thresholds, and routing policy before any work is generated.

02

Reviewer-grade queues

Annotation, disagreement review, gold checks, and low-confidence escalations live in the same operational surface.

03

Evidence-aware scoring

Domain rubrics evaluate source authority, modality, strictness, payload evidence, and metadata instead of treating every file alike.

04

Retrieval-ready exports

Teams leave with JSONL or CSV artifacts that carry provenance from request to reviewer decision.

Workflow

One control plane from request to retrieval.

1 Draft

Agent proposes schema, rubric, routing, and quality policy.

2 Approve

Operators edit and approve before assignments are generated.

3 Annotate

AI pre-labels support human decisions with visible evidence.

4 Adjudicate

Reviewers resolve disagreement, gold failures, and confidence risk.

5 Retrieve

Faceted search returns only assets matching authority and modality constraints.

Architecture

Neuro-symbolic data qualification, packaged as a product.

FacetLab uses archetypes, context-aware heuristics, faceted authority matrices, and terminal execution routes. For fMRI, that means pure retrieval of verified assets. For model evaluation, it means governed label collection and export. The same primitives scale to new modalities.
